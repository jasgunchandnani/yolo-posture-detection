## Basic Posture & Object Detection (YOLO + OpenCV)

This project uses YOLOv8 for real-time object detection and human posture
classification (standing, sitting, bending).

### Features
- Object & person detection
- Pose-based posture classification
- Image & video support
- Modular, scalable architecture

### Run Image
python run_image.py

### Run Video
python run_video.py
